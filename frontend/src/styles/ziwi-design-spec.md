## 2. 布局

### 2.1 整体结构

```
┌──────────────┬──────────────────────────────────────┐
│              │  顶部栏（标题 + 用户信息 + 操作）     │
│  侧边栏      ├──────────────────────────────────────┤
│  220px       │                                      │
│  深色背景    │  内容区                              │
│  #212529     │                                      │
│              │                                      │
└──────────────┴──────────────────────────────────────┘
```

- 侧边栏宽度：220px（折叠态 60px）
- 顶部栏高度：~50px
- 内容区背景：`#f5f6fa`
- 卡片背景：`#ffffff`，圆角 8px，阴影 `0 1px 4px rgba(0,0,0,0.06)`

### 2.2 间距

- 卡片内边距：16px
- 卡片间距：16px
- 内容区边距：24px
- 表单项间距：16px

## 3. 组件规范

### 3.1 按钮

| 类型 | 样式 | 使用场景 |
|:-----|:-----|:---------|
| **主按钮** `.btn-primary` | 背景 `#0d7377`，白色文字 | 主要操作（新建/保存/提交） |
| **描边按钮** `.btn-outline` | 白色背景，`#0d7377` 边框 | 次要操作（取消/筛选） |
| **文字按钮** `.btn-link` | 无边框，`#0d7377` 文字+下划线 | 轻量操作（编辑/查看详情） |
| **危险按钮** `.btn-danger` | 背景 `#ef4444` | 删除/吊销等危险操作 |

### 3.2 KPI 指标卡片

```
┌────────────────────────┐
│ █ 今日产量              │  ← 标题（12px, #555）
│ █ 1,286    件           │  ← 数值（20px 加粗）+ 单位
│ █ ↑ 较昨日 +12%         │  ← 趋势（绿色上/红色下）
└────────────────────────┘
```

- 左侧 3px 竖条：`#0d7377`
- 圆角 8px，白色背景，卡片阴影

### 3.3 表格

- 表头背景：`#f8fafc`，文字 `#555`，500 字重
- 行高：42px（10px padding 上下）
- 行悬浮：`#f8fafc` 背景
- 底部边框：1px solid `#e2e8f0`
- 状态标签：圆角 10px 胶囊，根据不同状态着色

### 3.4 表单

- 输入框：1px 边框 `#e2e8f0`，圆角 4px
- Focus 态：边框变为 `#0d7377`
- 标签：12px，`#555`，500 字重，距输入框 6px

### 3.5 弹窗（Modal）

- 遮罩：黑色 40% 透明度
- 弹窗：白色背景，圆角 8px，最小宽 480px
- 底部操作区：上边框 `#e2e8f0`，右对齐

### 3.6 标签页（Tabs）

- 活性态：`#0d7377` 底部 2px 实线 + 文字颜色
- 非活性态：`#555` 文字
- 底部 1px `#e2e8f0` 分隔线

### 3.7 搜索栏

- 输入框：宽 280px，圆角 4px
- Focus 态：边框 `#0d7377`
- 与后续操作按钮间距 12px

## 4. 登录页

- 背景：135deg 渐变 `#0d7377` → `#0a5c5f` → `#0a4a4d`
- 登录卡片：居中，白色背景，圆角 8px
- 输入框 Focus：边框 `#0d7377`
- 登录按钮：`#0d7377` 背景

- 弹窗在移动端全屏显示

## 7. CSS 变量

所有设计 token 定义在 `src/styles/ziwi-variables.css` 中，通过 `var(--ziwi-xxx)` 引用。

```css
/* 使用示例 */
.btn-submit {
  background: var(--ziwi-primary);
  color: var(--ziwi-text-white);
  border-radius: var(--ziwi-radius-sm);
}
```

---

## 8. 文件引用方式

### 客户前端（Vue 3 + Vant）
在 `main.ts` 或 `App.vue` 中引入：
```typescript
import '@/styles/ziwi-variables.css'
```
Vant 主题覆写同样在全局样式中完成。

### 管理门户（Vue 3）
在 `App.vue` 中引入：
```typescript
import '@/styles/ziwi-variables.css'
```
所有组件直接使用 `.btn-primary` 等通用类名。
