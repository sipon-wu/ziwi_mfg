### 指令: "现在处理：创建统一的依赖注入函数 get_tenant_repo()，自动从当前用户提取 tenant_id 并注入 repo。将所有 API 路由改为使用该依赖进行 repo 创建"

## 完整租户隔离链

```
get_current_user → 提取 tenant_id
       ↓
get_tenant_repo() → repo.set_tenant_id(tenant_id)
       ↓
repo.execute() → WHERE tenant_id = :_tenant_id  ← P0 修复
repo.query()    → WHERE tenant_id = :_tenant_id  ← 基类已有
```

受保护路由的 UPDATE/DELETE/query 全部被 tenant_id 限定，形成完整的多租户行级隔离。
