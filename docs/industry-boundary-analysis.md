# 知微 ziwi SaaS 行业边界分析

> 基于 SAP 行业分类方法论，明确知微平台的目标行业边界与排除原则

---

## 1. SAP 行业分类体系参考

SAP 将全部行业划分为 6 大集群，覆盖 25+ 垂直行业：

| 集群 | 典型子行业 | 生产模式 | 知微是否覆盖 |
|:----:|-----------|:--------:|:-----------:|
| **离散制造** | 汽车零部件、机械设备、电子组装、家电制造 | 工单驱动 + 工序装配 | ✅ **主攻** |
| **流程制造** | 化工、制药、食品饮料、钢铁、水泥 | 配方驱动 + 连续/批次 | ⚠️ 兼顾核心需求 |
| **消费品** | 服装纺织、鞋帽箱包、快消品 | 季节驱动 + SKU 高度分裂 | ❌ 排除 |
| **能源与资源** | 石油天然气、矿业、电力 | 资产密集 + 连续流程 | ❌ 排除 |
| **金融服务** | 银行、保险、证券 | 非制造业 | ❌ 排除 |
| **公共服务** | 政府、教育、医疗 | 非制造业 | ❌ 排除 |

## 2. 行业覆盖边界矩阵

### ✅ 主攻行业（IN — 产品设计核心覆盖）

| 行业 | 典型产品 | 生产特征 | 关键需求 | 知微覆盖度 |
|:----:|---------|---------|---------|:----------:|
| **汽车零部件** | 传动轴/发动机支架/制动系统 | 多品种小批量、工序级管控、批次追溯 | 工艺路线可视化、SPC、安灯响应 | ⭐ **重点** |
| **机械装备** | 泵阀/减速机/机床 | BOM多层级、装配复杂、外协加工 | 工艺路线管理、工时统计、质量追溯 | ⭐ **重点** |
| **电子组装** | PCB/控制器/传感器 | SMT贴片、静电防护、元器件追溯 | 防错料、工单流转、质检联动 | ⭐ **重点** |
| **金属加工** | 精密加工件/模具 | CNC加工、热处理、表面处理 | 设备监控、刀具管理、工序报工 | ✅ 覆盖 |
| **家电制造** | 小家电/白电配件 | 流水线装配、零部件多 | 工单管理、报工统计、品质检验 | ✅ 覆盖 |

### ⚠️ 兼顾行业（IN — 核心需求覆盖，行业特需暂不实现）

| 行业 | 覆盖原则 | 实现范围 | 不实现范围 |
|:----:|---------|---------|-----------|
| **化工** | 批次管理+配方 | 批次追溯、物料管理、质量检验 | DCS深度集成、连续流程控制、联副产品管理 |
| **制药** | 批次+质检 | 批次记录、检验流程、物料追溯 | GMP合规、电子批记录(EBR)、审计追踪、21 CFR Part 11 |
| **食品饮料** | 配方+批次 | 配方管理、保质期预警、原料追溯 | HACCP合规、口味测试、感官评价 |

### ❌ 排除行业（OUT — 当前版本不覆盖）

| 行业 | 排除原因 | 关键差异 |
|:----:|---------|---------|
| **服装纺织** | 生产模式完全不同 | 季节驱动 + SKU高度分裂 + 尺寸矩阵 + 面料色差管控 |
| **鞋帽箱包** | 生产模式完全不同 | 裁断→针车→成型三阶段 + 工人计件工资为主 |
| **石油天然气** | 资产密集非离散制造 | 连续流程 + 管道运输 + EHS合规 + 设备完整性管理 |
| **矿业** | 非工厂制造场景 | 采掘作业 + 地质建模 + 矿车调度、非工序管理 |
| **电力能源** | 非离散制造 | 发电调度 + 电网管理 + 交易结算、非工单驱动 |
| **航空航天** | 合规要求极高 | AS9100 + 适航管理 + 序列号逐件追溯 + 供应商管控复杂 |
| **半导体** | 工艺极度精密 | 晶圆级追溯 + 纳米精度工艺控制 + 黄光区管理 |

## 3. 排除行业的判决理由（核心逻辑）

### 为什么排除服装纺织？

| 维度 | 知微设计（离散制造） | 服装纺织需求 | 差异 |
|:----|:-------------------|:------------|:----:|
| **生产模式** | 工单驱动 + 工序装配 | 季节驱动 + 款号批量 | ❌ |
| **物料管理** | BOM树状结构 | 面料+辅料+尺寸矩阵（如S/M/L/XL） | ❌ |
| **工艺路线** | 前后工序串行+并行 | 裁断→缝制→整烫→包装，线性 | ❌ 虽简单但排产维度不同 |
| **排产逻辑** | 设备产能+工时 | 工人技能+流水线平衡+节拍 | ❌ |
| **质量管控** | 首检+巡检+SPC | 面料色差+缝制平整度+尺寸公差 | ❌ |
| **行业合规** | 无特殊合规 | AQL抽检标准、色牢度检测 | ❌ |
| **市场规模** | — | 较大但需求差异大 | 投入产出比不划算 |

### 为什么排除制药（但化工可兼顾）？

| 维度 | 知微可覆盖 | 制药额外需求 | 实现成本 |
|:----|:----------|:------------|:--------:|
| 批次追溯 | ✅ 基础批号 | 电子批记录(EBR) + 物料平衡 | ★★★★ 高 |
| 检验流程 | ✅ 检验单 | 加密电子签名 + FDA 21 CFR Part 11 | ★★★★ 高 |
| 设备管理 | ✅ 设备台账 | 校验到期预警 + 清洁验证 | ★★★ 中 |
| 审计追踪 | ❌ | 全方位数据不可篡改 + 双人复核 | ★★★★ 高 |
| 环境监控 | ❌ | 洁净区A/B/C/D级环境 + 在线粒子监控 | ★★★ 中 |

> **结论**：制药需要 4 项中高成本的合规增强，且在无客户明确需求时不应提前投入。

## 4. 知微的目标行业画像

### 理想客户画像

```
企业性质：中小型制造企业（100-500人）
工厂类型：离散型制造（车间+产线+设备+工人）
生产模式：多品种小批量、按订单生产（MTO）
管理痛点：工单流转靠纸质、设备状态靠巡检、质量数据靠手填
数字化程度：Excel + 纸质单据为主，急需低门槛 SaaS
地理位置：国内制造业密集区域（长三角/珠三角）
```

### 目标行业优先级排序

```
P0 ─── 汽车零部件 ─── 机械装备 ─── 电子组装
       （行业标杆，最先切入）
P1 ─── 金属加工 ─── 家电制造 ─── 模具制造
       （横向扩展，轻度适配）
P2 ─── 化工 ─── 食品 ─── 制药 ─── 医疗器械
       （流程型/合规型，需行业插件）
排除    服装/鞋帽/石油/矿业/航空航天
       （生产模式不匹配或合规投入过高）
```

## 5. SAP 的启示：知微的适配策略

### SAP 怎么做

- **公有云**：预配置行业最佳实践模板 + 参数化配置
- **本地部署**：允许完整的代码级定制（ABAP）

### 知微怎么做（轻量版）

```
行业适配策略（Layered Approach）:

Layer 1: 核心引擎（不变）
  ├─ 工单管理 / 报工 / 质检 / 设备管理
  ├─ 工艺路线 / 工序编排 / 工作中心
  └─ 组织权限 / 数据作用域
  
Layer 2: 行业参数包（可配置）
  ├─ 工序类型枚举（离散 vs 流程）
  ├─ 质检规则模板（离散：首检+抽检；流程：批次取样）
  ├─ 报工模式（离散：按件报工；流程：按批次报工）
  └─ 排产偏好（离散：工序级；流程：流水线级）

Layer 3: 行业插件（独立模块，按需加载）
  ├─ 配方管理（化工/食品）
  ├─ 电子批记录（制药）
  ├─ SPC控制图（质量要求高行业）
  └─ 设备IoT监控（设备密集型行业）
```

## 6. 边界结论

### 一句话边界

> **知微 SaaS = 面向中小型离散制造企业的轻量 MES 平台，以汽车零部件/机械装备/电子组装为核心战场，兼顾化工/食品的批次管理需求，排除服装纺织/航空航天/石油矿业等生产模式不匹配的行业。**

### 当前版本（Phase 1-2）边界

| 维度 | 边界 |
|:----|:----|
| **行业** | 汽车零部件、机械装备、电子组装 |
| **生产模式** | 离散型制造（工单驱动+工序装配） |
| **企业规模** | 100-500人 |
| **排产复杂度** | 手动排产为主，系统辅助 |
| **合规要求** | 无行业专项合规 |
| **第三方集成** | 无强制依赖，API预留 |
| **IoT集成** | 能碳数据为基础，设备监控为Phase 4 |

### 明确不做的

| 不做 | 理由 |
|:----|:----|
| 服装纺织行业 | 生产模式不匹配（季节驱动+SKU分裂） |
| 航空航天/医疗器械合规 | 专项合规成本过高，有客户需求时再评估 |
| GMP/FDA 合规认证 | 制药行业的独立赛道，非当前目标 |
| DCS 深度集成 | 流程型行业基础设施依赖，可复用能碳IoT |
| APS 高级排程 | 算法复杂度高，当前手动排产即可 |
