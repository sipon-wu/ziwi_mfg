> 对照 `ziwicloud.xmind` 中 MES_生产模块 / TPM_设备管理 / QMS_品质模块 / LiMS_实验室 / Andon_安灯 / 生产计划及APS / 数据采集 / 看板 的粒度，逐项评审当前 `product-functional-specification.md` 的覆盖度
> 日期：2026-06-18（M07/M08/M09 更新于 2026-06-19）

## 1. 生产计划及APS

### XMind 粒度

```
排产计划
  ├─ 排产
  │   ├─ 生成：根据实时任务单/交期/型号产量/库存量/设备工况排出投产队列→甘特图
  │   ├─ 查看：排产甘特图(日/周/月)、负荷甘特图(日/周/月)
  │   ├─ 编辑：插单(依线别排列队列、前后拖拽、已投产不可拖动)
  │   └─ 配置：工厂日历(排班计划)、标准节拍调整(按瓶颈工序倒序)、优化调整(产量/设备寿命/能耗)
  └─ 派工单
      ├─ 新建：选任务单→指定线别(班组)→指定数量/完成时间→预览高亮→确认创建
      ├─ 查看：列表(字段:派工单号/产品/客户/数量/各状态完成量/责任人/日期/状态)
      └─ 状态：已报开工→已确认开工→已全检(下线)→已入库(完成)→已取消
```

### 当前规格书差距

| 维度 | XMind 要求 | 当前规格书 | 差距 |
|:----|:----------|:----------|:----:|
| 排产生成依据 | 实时任务单+交期+库存量+设备工况 | M07-07 排产生成：支持正向/倒推排产，基于任务单交期/型号产量/库存量/设备工况 | ✅ |
| 甘特图类型 | 排产甘特图+负荷甘特图(日/周/月) | M07-08 排产甘特图查看(日/周/月)，M07-09 负荷甘特图查看(日/周/月) | ✅ |
| 插单交互 | 依线别排列队列+前后拖拽+已投产不可拖 | M07-10 排产编辑-插单(按线别排列+已投产不可拖动)，M07-11 排产拖拽调整(前后拖拽+产能约束) | ✅ |
| 节拍调整 | 按瓶颈工序倒序排列+调整节拍 | M07-12 排产配置-节拍调整(按瓶颈工序倒序排列各工序节拍，支持逐工序调整) | ✅ |
| 优化目标 | 产量最大化/设备寿命/能耗最小 | M07-12 排产配置-优化策略(产量最大化/设备寿命优先/能耗最小化三种策略) | ✅ |
| 派工单 | 独立单号+线别+派工数量+全检量+包装量+5种状态 | M07-13~M07-18 完整派工单模块：新建/列表/详情/报开工/确认开工/全检入库取消，5种状态流转 | ✅ |

## 2. MES_生产模块

### XMind 粒度

```
概览(看板)
  ├─ 通知汇总(任务/报警/系统审批)
  ├─ 当日生产实时数据(仪表盘)
  ├─ 人员出勤实时(应到/实到/休假/旷工)
  ├─ 设备工况实时(仪表盘/二维图/孪生入口)
  ├─ 实时不良率
  └─ 物料状况实时(齐套/缺料)

生产任务单
  ├─ 新建(对接ERP下发)
  ├─ 查看：列表(14个字段)、详情(全部含派工单/工艺路线/工单)
  └─ 状态：已排产→已开工→已全检→已入库→已取消

排产计划 → (见第1节)

派工单 → (见第1节)

报工记录
  ├─ 当日报工(实时)：按派工单(派工单号/开工/全检/包装/作业员/PQC/FQC)
  │                         按产线/工作中心(一线/二线/包装中心 维度)
  └─ 往日报工记录：切换日期+起止时间筛选

生产报表(日/周/月/半年/年)
  ├─ 维度：产品型号/线别/车间别
  └─ 指标：产量/达成率/不良率/总耗能 + 导出

配置
  ├─ 车间-线别/工作中心-工站/工序(关联设备)
  │   ├─ 设备标准产能/节拍(与TPM同步)
  │   └─ 开机预备工时损耗(与TPM同步)
  ├─ 工作班组(对应车间-线别)
  ├─ 工作日历
  ├─ 产品型号 → 工艺路线 → BOM
  ├─ 补数原因项
  └─ 权限及审批流程
```

## 3. TPM_设备管理模块

### XMind 粒度

```
概览
  ├─ 通知汇总(维修/保养/报警/审批)
  ├─ 设备工况实时数据(仪表盘/二维图/孪生入口)
  └─ 统计分析日/周/月简报

任务中心
  ├─ 维修任务：新建/列表/详情/7种状态(提交→确认→维修中→完成→取消→关闭)
  │            字段:任务号/设备名称/编号/位置/报修人/故障说明/提交时间/要求完成/责任人
  ├─ 保养任务：新建/字段(含保养级别:日/周/月/季/半年/年度)/延期流程/6种状态
  └─ 采购任务(备品备件)：安全水位触发/请购/到库验收/领用报废

维修与保养
  ├─ 维修记录：详细字段(异常现象/故障描述(含附件)/原因/方案/经验总结)
  └─ 保养记录：同上

设备
  ├─ 生产设备：新建/列表/详细字段(20+字段含稼动率/可生产产品/总功率/供应商/责任人)
  ├─ 工装/模具：同上
  ├─ 备品备件：编号/品名/用途/关联设备/在库量/安全量/属性(耗材/易损件)/请购/领用(直接/换领)
  └─ 基建设备：锅炉/空调等

配置
  ├─ 团队：维修小组(对应设备+级别)、维护小组(对应日/周/月...)
  ├─ 设备分类：设备/模具/工装/仪器
  ├─ 故障项：常见故障明细+故障级别
  ├─ 保养项：配置保养任务(模板) → 选择设备→日/周/月/半年/年保养表→指定责任人
  │          配置保养类型/频率(全局)：日/周/月/季/半年/年度→打勾
  │          新增/编辑保养项 → 上传作业指导书
  └─ 备品备件：设置类别+安全库存量
```

### 当前规格书差距

| 维度 | XMind 要求 | 当前规格书 | 差距 |
|:----|:----------|:----------|:----:|
| 概览 | 通知+工况+统计分析简报 | 无 | ❌ |
| 任务中心 | 维修/保养/采购三类任务独立管理 | 仅有"发起保养/维修" | ❌ |
| 维修详细字段 | 异常现象+故障描述(附件)+原因+方案+经验 | 无 | ❌ |
| 保养级别 | 日/周/月/季/半年/年度 + 延期流程 | 无 | ❌ |
| 备品备件 | 安全水位/请购/领用/换领/直接领用 | 仅有基础spare_parts | ❌ |
| 故障项配置 | 常见故障明细+故障级别 | 无 | ❌ |
| 保养项模板 | 选择设备→多级保养表→指定责任人→上传SOP | 无 | ❌ |
| 团队 | 维修小组+维护小组(按级别对应设备) | 无 | ❌ |
| 工装/模具 | 独立管理(非设备) | 无 | ❌ |

## 4. QMS_品质模块

### XMind 粒度

```
概览
  ├─ 通知汇总(检验任务/车间质量报警/客户退货/判定非ACC)
  ├─ 当日质量实况(不良率+各产线柱状图)
  ├─ 周不良率/月不良率
  └─ 月度TOP5

检验单
  ├─ 新建：由移动屏终端执行(首检/巡检/抽检三种模式)
  ├─ 列表字段：检验单号/分类(首检/巡检/抽检)/产品/工单/检查员/日期/判定(ACC/REJ/UAI)/特采人/备注
  └─ 详情：上述+订单号/客户/型号/特殊要求

测试申请表(对接实验室)

合格证
  ├─ 成品抽检ACC+UAI结果筛选
  ├─ 查看合格证+打印(支持多选)
  └─ 打印合格证

记录与统计(日/周/月/自选起止)
  ├─ 按产线/按派工单/按订单/按客户

质控点配置(基础数据)
  ├─ IQC：启用→抽样标准→配置材料明细(每种材料配置检验项目+测试项目)
  ├─ IPQC：启用→抽样标准→配置巡检频次(全局1H/2H/自定义)
  │         → 配置烫带(检验项目+节拍)
  │         → 配置各产线各工序(检验项目+测试项目+独立节拍)
  ├─ FQC：启用→抽样标准→检验项目+测试项目
  └─ OQC：启用→抽样标准→检验项目+测试项目

配置
  ├─ 检验团队
  ├─ 检验标准(国标/企标)+判定准则(轻微*n/严重*n→ACC/REJ/UAI)
  ├─ 检验项目(字段:序号/编号/类目/方式/工具/单位/公差/环境/SIP)
  │   目视适用：编号/项目/方式/环境/不合格项/SIP
  │   测量适用：编号/项目/方式/工具/单位/公差/环境/不合格项/SIP
  │   新增：检查方法/仪器工具+场所/标准值+公差/环境/不合格描述(严重/轻微)/记录形式/附件
  └─ 测试项目(转实验室模块，同步)
```

| 维度 | XMind 要求 | 当前规格书 | 差距 |
|:----|:----------|:----------|:----:|
| 检验单模式 | 首检/巡检/抽检三种模式 | M10-01~M10-02, M10-12 三种模式已定义 | ✅ |
| 判定类型 | ACC/REJ/UAI(特采)+特采人 | M10-04, M10-15 含完整UAI特采审批 | ✅ |

| 检验项目 | 详字段(方式/工具/单位/公差/环境/不合格项/SIP/附件) | M10-34 完整字段结构 | ✅ |
| 判定准则 | 轻微*n/严重*n → ACC/REJ/UAI 规则引擎 | M10-33 判定规则引擎 | ✅ |

## 6. Andon_安灯模块

### XMind 粒度

```
概览
  ├─ 图示：实时报警(工序点阵图/二维表/孪生)颜色标示
  ├─ 数显：呼叫次数/正在报警数
  ├─ 柱状图：当日故障总时长/各线总时长
  ├─ 分类百分比：当日/当月
  ├─ 响应超时统计：平均/超10/20/30分钟比例
  └─ 处理时长统计：平均/超1h/2h/3h/1d比例

  ├─ 字段：序列号/时间/类型/发生源/订单/发生/响应/处理时间/状态

  ├─ 呼叫次数(日月)

配置
  ├─ 异常大类(每个类型可独立启用/禁用)
  │   └─ 通知对象设置+响应时段配置(正常/延时/严重延时)+处理时长配置
  ├─ 消息通道(看板/广播/微信/钉钉/短信/邮箱)

```
